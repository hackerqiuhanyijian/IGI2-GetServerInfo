# IGI2 Server Status Query Tool

This is an IGI2 game server status query tool.

## Features
- Query IGI 2 game server status via UDP protocol implemented in C++.
- Parse detailed server information including server name, player count, current map, and more.
- Includes complete Visual Studio 2026 project files for out-of-the-box usage.

## Usage
1. Clone or download this repository.
2. Open `IGI2-GetServerInfo.sln` with Visual Studio 2026.
3. Build and run the project to query IGI 2 server details.

## Important Note
**Users must update `IGI2-GetServerInfo.vcxproj` after downloading to ensure successful compilation.**

## Compilation
- Recommended: Visual Studio 2026
- Build Mode: Release /MT (for standalone executable)
